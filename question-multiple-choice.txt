4. What will be the output?
   t = (1, 2, 4, 3, 8, 9)
   [t[i] for i in range(0, len(t), 2)]

   A) [2, 3, 9]
   B) [1, 2, 4, 3, 8, 9]
   C) [1, 4, 8]
   D) (1, 4, 8)

Answer:  C
Explanation: List comprehension in the second line and in range using 2 as third params, it will return 0, 2, 4
